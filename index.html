Application Web, Mobile et API

Application de réservation de DVD

Portant le nom de RFTG, cette application permet de mettre en pratique de nombreuses notions apprises dans le cadre du BTS SIO. Elle permet la réservation de DVD dans un parc d'attraction avec une partie applicative mobile pour les clients, une partie web pour les administrateurs ainsi qu'une API REST connectée à une base de données inspirée du modèle Sakila.

En développement

Détails du projet

  • Technologies utilisées : PHP, MySQL, Java, XML, REST API, Android Studio
  • Durée : Projet en cours
  • Modules : Gestion des réservations, authentification, interface mobile et web
  • Objectif : Créer un système complet de gestion de DVD pour un parc à thème

Défis techniques : communication entre l'application mobile et le serveur via une API REST sécurisée, gestion multi-interface (web + mobile) et synchronisation des données avec la base MySQL.
