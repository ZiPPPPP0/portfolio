À compléter

Présentation du Projet

RFTG est une application complète permettant la gestion et la réservation de DVD dans un parc d'attraction. Le projet met en pratique de nombreuses notions apprises dans le cadre du BTS SIO.

Architecture du Projet

L'application se compose de trois parties principales :

  • Application Mobile (Android) - Interface client pour consulter et réserver des DVD
  • Interface Web - Panneau d'administration pour gérer le catalogue et les réservations
  • API REST - Backend sécurisé pour la communication entre les différentes interfaces

Technologies Utilisées

PHP MySQL REST API Java Android Studio JWT

Fonctionnalités Principales

  • Consultation du catalogue de DVD avec filtres de recherche
  • Système de réservation en temps réel
  • Gestion des utilisateurs et authentification sécurisée
  • Interface d'administration pour la gestion du stock
  • Synchronisation automatique entre mobile et serveur

Défis Techniques

Communication Mobile-Serveur

Mise en place d'une API REST sécurisée avec gestion des tokens d'authentification et validation des requêtes.

Synchronisation des Données

Gestion de la cohérence des données entre l'application mobile et le serveur, avec gestion des conflits et mise à jour en temps réel.

Gestion Multi-Interface

Architecture permettant à plusieurs interfaces (web et mobile) d'accéder aux mêmes données de manière cohérente.
